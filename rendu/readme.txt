# ProjetACVL - Ensimag 2A Apprentis
Projet de conception - implémentation d'un logiciel modélisant des diagrammes d'états transition
Groupe Bianchini Thomas - Couret Nathanael - De Kimpe Arthur - Hagenbourger Maxime

Ce projet est basé sur Apache Maven, pour l'importer dans un IDE, il convient de l'importer comme tel. 

Pour lancer l'application, téléchargez l'archive et l'extraire complètement. Lancer ensuite l'exécutable app.jar après l'avoir rendu exécutable (nécessaire suivant le système d'exploitation).
